Готовый статический сайт для GitHub Pages.

## Что внутри

- `index.html` — презентация курса в формате сайта/слайдов

## Как опубликовать на GitHub Pages

1. Создай новый репозиторий на GitHub
2. Загрузите в него файл `index.html`
3. Зайдите в `Settings` → `Pages`
4. В блоке **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (или `master`) / `/root`
5. Сохраните настройки
6. Через 1–3 минуты сайт будет доступен по ссылке GitHub Pages

## Как обновлять

Если захочешь изменить тексты или дизайн, достаточно заменить `index.html` в репозитории новой версией.
